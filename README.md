# yuntu
云图的一键部署项目。  
*神州数码云基地出品*

## 环境要求
docker  
docker-compose

## 运行方法
```
git clone https://github.com/keeperkwok/yuntu.git  
cd yuntu
docker-compose up -d 
```
## 访问地址
```
http://localhost:8888
```
